Pokedex

Aqui temos uma Pokedex que apresenta informações sobre pokemons. Essas informações são obtidas através do consumo da API pokeapi.co.
Para utilizá-la basta digitar o nome ou o número de um pokemon no campo e depois clicar em Enter para visualizar.

Desenvolvido com:
HTML, CSS, JavaScript

Kontacts

O Kontacts trata-se de uma agenda virtual particular, e para utilizá-la o usuário deve primeiro se cadastrar e em seguida fazer o login e assim gerenciar seus contatos, isto inclui: cadastrar, apagar, alterar e listar todos os dados. Esta é uma das minhas aplicações mais completas onde é composta por três páginas e os dados são armazenados e requisitados por um servidor remoto.

Desenvolvido com:
HTML, CSS, JavaScript, React
Acesse o Kontacts clicando aqui!!
